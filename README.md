


# Hex Siege 🏹

Hex Siege is a 3D mobile tower defense game built in Unity with a hex grid environment system. The central points of the game development are : high optimization , design, and constantly changing and evolving environments.

The game implements "Gradient Atlas Texturing" a highly efficient technique where all materials are derived from a single low-resolution texture atlas.
